# 🧪 🧰 KIT COMPLETO DE PROMPTS PARA QA

## 🧩 1. Análise de requisito (ANTES de testar)

```id="qa001"
Atue como um QA sênior.

Analise o seguinte requisito:

[cole aqui]

Identifique:
- possíveis ambiguidades
- pontos de risco
- cenários não cobertos
- dúvidas que deveriam ser feitas ao time

Responda de forma estruturada.
```

👉 Use isso antes de qualquer teste (isso já te destaca no time)

---

## 📋 2. Criação de cenários de teste completos

```id="qa002"
Você é um QA experiente.

Crie cenários de teste para:

[funcionalidade]

Inclua:
- casos positivos
- casos negativos
- edge cases
- critérios de aceitação implícitos

Organize em formato de lista clara.
```

## ⚠️ 3. Descobrir bugs antes de acontecer (mentalidade sênior)

```id="qa003"

[funcionalidade]

```
