# 🧪 📋 Etapas do projeto

## 1. Criar uma funcionalidade fictícia

Exemplo:

👉 Sistema de login com:

* email
* senha
* recuperação de senha

---

## 2. Análise com IA

Use prompt:

```text
Atue como QA sênior.

Analise este sistema de login:

[email + senha + recuperação]

Liste riscos, falhas e dúvidas.
```

📌 Salve em:
`docs/test-plan.md`

---

## 3. Criar cenários de teste

```text
Crie cenários de teste para login.

Inclua positivos, negativos e edge cases.
```

📌 Salve em:
`docs/test-cases.md`

---

## 4. Criar bug reports

Exemplo:

```text
Usuário não consegue logar com senha válida após reset.
```

Use IA:

```text
Crie bug report profissional.
```

📌 Salve em:
`bugs/bug-reports.md`

---

## 5. Automação de testes

Escolha uma ferramenta:

* Cypress
* Playwright (recomendado 🚀)

Use IA:

```text
Crie teste automatizado para login válido e inválido.
```

📌 Salve em:
`automation/`

---

## 6. Testes de API

Simule endpoint:

```bash
POST /login
```

Use IA:

```text
Crie testes de API com validações completas.
```

---

## 7. Documentar uso de IA

Crie:

📄 `docs/prompts.md`

Coloque:

* para que servem
